# FMAvailableResult

Result of `fm available`.

## Inherits

- [FMResult](FMResult.md)

## Computed properties

| Property    | Type    | Description                                                |
| ----------- | ------- | --------------------------------------------------------- |
| `available` | Boolean | `True` if the checked model is reported as available.     |

The raw status line is available through the inherited `text` property.
